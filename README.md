##Pivot Class Scheduler

###Install dependencies
```bash
npm install
```

###Run
development:
```bash
npm start
```

production
```bash
npm run start:production
```

specifying API_URL and API_VERSION:
```bash
export API_URL=http://dev.trainwithpivot.com API_VERSION=1 npm start
```
by default API_URL=http://localhost:8080 and API_VERSION=1
